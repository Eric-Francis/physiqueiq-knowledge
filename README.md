# PHYSIQUEIQ Knowledge

## Mission

PHYSIQUEIQ Knowledge is the intelligence platform that powers every recommendation inside PHYSIQUEIQ.

The goal is to create the world's most trusted evidence-based AI fitness platform.

This repository will eventually contain:

- Exercise Library
- Nutrition Library
- Coaching Rules
- Progression Rules
- Recovery Rules
- Supplement Library
- Powerlifting Knowledge
- Bodybuilding Knowledge
- ICN Prep Knowledge
- Scientific Research Summaries
- Decision Engine Rules

## Principles

1. Every recommendation must be evidence-based.
2. Every recommendation must have a scientific source.
3. Every recommendation should be explainable.
4. The user should never manually build their own program.
5. PHYSIQUEIQ acts as the coach.

This repository is the foundation of the PHYSIQUEIQ Intelligence Platform.
